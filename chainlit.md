# TokenTriage Judge Demo

TokenTriage is an Agent FinOps control plane for production AI agents.

Use the scenario buttons to watch the full routing pipeline:

- Security Gateway
- Semantic Cache
- Triage Agent
- MCP Pricing and Benchmarks
- Policy Engine
- Model Dispatch
- Verifier
- Savings Receipt

The final answer includes the selected tier, modeled TokenTriage cost, always-frontier baseline, cache status, verifier result, and savings percentage.
